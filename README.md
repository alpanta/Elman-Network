# Elman Network - Billings System

Elman recurrent neural network implementation to predict Billings chaotic time series.

## Files
- `elman.m` — MATLAB implementation
- `elman.py` — Python implementation (NumPy + Matplotlib)

## Setup
```bash
python -m venv .venv
.venv/bin/pip install numpy matplotlib
.venv/bin/python elman.py
```
